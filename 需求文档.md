# 在线多人贪吃蛇游戏
### 技术栈
python fastapi做后台，前台使用vue
### 效果
编写一个可以多人同时在线的网页版贪吃蛇游戏
### 账号
可以注册账号，账号只需要用户名和密码就行
### 多人游玩
登录后，可以自己开主机，也可以加入别人的主机
主机可以决定什么时候开始游戏
在房间内，可以设定自己的emoji图标作为蛇头、可以选择颜色，可以组队（两个颜色一样的就是一队）
### PVP玩法
* 撞到别人就可以把别人吃掉
* 用WASD控制贪吃蛇上下左右移动
* 苹果会随机生成在地图内的任意地方
* 贪吃蛇吃掉苹果身子长度会增长
* 如果撞到墙壁或自己的身体死亡（5秒后复活，复活期间自己能看到倒计时）
* 会有特殊的苹果：
  * 加速
  * 增长
  * 无敌x秒（3、5、10秒）
  * 时间减慢（所有人慢动作5秒）
